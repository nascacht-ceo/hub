# Reflection

The `TypeService` class provides a mechanism to build types at runtime that implement standard patterns and practices.


|Term|Description|
|-|-|
|Solution|A collection of classes used to solve business domain specific problems. Examples might include `InsuranceClaim`, `CreditCardApplication`, or `DoctorsVisit`.|
|ClassName|A data model used by a `Solution`. Examples might include `Claim`, `Application`, or `Visit`.|
